PRODUCT NAME
  CalcMint

TAGLINE
  Plan . Save . Grow

ONE-LINE PITCH
  A complete personal-finance toolkit for India: 12 calculators, expense /
  loan / subscription tracking, document vault, and goal planning - all
  offline on your phone.

CATEGORY
  Finance / Personal Finance / Productivity

TARGET AUDIENCE
  Indian users (salaried professionals, investors, home-loan seekers, students)
  who need quick financial estimates and lightweight money tracking without

------------------------------------------------------------------------------
APP STRUCTURE
------------------------------------------------------------------------------

  5 bottom tabs:
    1. Dashboard - overview: spent this month, EMI/MO, Subs/MO, goals,
                   quick-actions row (Expense, Loans, Split, Subs, Docs)
    2. Tools     - calculator grid + search
    3. Expenses  - monthly expense logger with category breakdown donut chart
    4. Vault     - PIN-locked secure storage for notes / credentials
    5. More      - settings, backup, lock setup, account info

  The Tools tab opens a stack of 12 calculator screens.
  Shared flows (Settings, Loans, Receipts, Subscriptions, Split Groups,
  Goals) live in the root stack and are reachable from any tab.

------------------------------------------------------------------------------
CALCULATORS (12 TOTAL)
------------------------------------------------------------------------------

  1.  Home Loan       - EMI + total interest breakdown, pre-payment scenarios
  2.  EMI             - generic loan repayment planner
  3.  SIP             - systematic investment plan future value
  4.  Lumpsum         - one-time investment growth with CAGR
  5.  Fixed Deposit   - FD maturity with monthly/quarterly/annual compounding
  6.  Recurring Dep.  - RD monthly growth and maturity
  7.  PPF             - 15-year PPF projection at current 7.1% rate
  8.  Tax Savings     - Old vs New tax regime comparison (Indian income tax)
  9.  Goal Planner    - reverse-calculate the monthly SIP needed for a target
  10. Budget          - 50/30/20 budget allocation
  11. Retirement      - retirement corpus projection
  12. Unit Converter  - common unit conversions

------------------------------------------------------------------------------
KEY FEATURES
------------------------------------------------------------------------------

  - 100 percent offline. No accounts, no login, no network calls for app data.
  - Indian currency formatting (lakh / crore notation).
  - Persists last-used calculator and last-used inputs per calculator.
  - Donut and bar charts for visual breakdown of results.
  - Dashboard with at-a-glance month spend, EMI / subscriptions, and goals.
  - "Continue where you left" shortcut on the Tools tab.
  - In-app search across the 12 calculators.
  - Subscription tracker with monthly reminders.
  - Loan tracker with monthly EMI reminders.
  - Split groups for shared expenses.
  - Receipts / Documents capture (Bill, Invoice, Loan paper, Warranty, etc.)
    with warranty-expiry reminders.
  - Goal tracking and expense logging persisted locally.
  - PIN-protected Vault for sensitive notes / credentials.
  - App-wide lock with PIN setup screen.
  - Encrypted on-device backup / restore (AES via crypto-js, exported through
    expo-sharing).

------------------------------------------------------------------------------
SECURITY / LOCK MODEL
------------------------------------------------------------------------------

  - 4-digit PIN configured in LockSetup; same PIN unlocks the Vault.
  - Vault uses an in-app PIN keypad only. The native Android / iOS device
    credential popup (LocalAuthentication) is NOT invoked during vault
    unlock - this avoids double-prompting on Android.
  - PIN hash and vault keys live in expo-secure-store.

------------------------------------------------------------------------------
PRIVACY POSTURE
------------------------------------------------------------------------------

  - All financial data (inputs, goals, expenses, loans, subscriptions,
    vault entries, documents) stays on-device via AsyncStorage / secure-store
    / local filesystem. Nothing is uploaded to a server (no backend exists).
  - Backups are encrypted locally and only shared via the user-initiated
    OS share sheet.
  - The only third-party SDK is Google Mobile Ads, which Google uses for
    ad serving. Ad requests use `requestNonPersonalizedAdsOnly: true` by
    default in this build.
